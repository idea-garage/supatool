# Changelog

All notable changes to this project will be documented in this file.

## [0.0.1] - 2025-02-09
- Initial version
- Support for dynamic schema name retrieval

## [0.0.2] - 2025-02-18
- Support for CRUD generation for views

## [0.0.3] - 2025-02-19

- Overwrite confirmation with `--force`/`-f`.
